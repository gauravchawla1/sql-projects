# SQL Practice Project

This repository contains a complete SQL practice environment with:
- A sample company database schema
- Insert scripts with realistic data
- Basic, intermediate, and advanced SQL queries
- An advanced business-style challenge

Designed as a hands-on SQL portfolio demonstrating proficiency in:
Joins, Aggregations, Subqueries, Window Functions, Ranking, and Data Analysis.

This serves as verifiable proof of SQL experience for assessments and hiring.
